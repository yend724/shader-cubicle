# Shader Cubicle

This is the shader practice project using [Three.js](https://threejs.org/).

## Install

```shell
$ cd shader-cubicle
$ yarn install
```

## Run locally

Start dev server.

```shell
$ yarn dev
```

## Build

Builds the project for production to the dist folder.

```shell
$ yarn build
```

## Preview

Locally preview production build.

```shell
$ yarn preview
```
